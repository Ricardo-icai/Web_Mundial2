# Sistema de agentes - WorldCup Fan Planner 2026

Este proyecto se organiza con cuatro agentes principales:

1. Experto en Dominio
2. Desarrollador
3. Quality Assurance
4. Auditor

## Flujo de trabajo recomendado

1. Experto en Dominio define requisitos y criterios.
2. Desarrollador implementa la funcionalidad.
3. Quality Assurance prueba los flujos funcionales y errores.
4. Auditor revisa arquitectura, seguridad y coherencia.
5. Desarrollador corrige hallazgos.
6. Auditor valida version final.
